Todo App (React + Vite + Tailwind)

A fast and minimal Todo application built with React, Vite, and Tailwind CSS.
Users can add and delete tasks with a clean UI and smooth state management using React Hooks. The project is lightweight, easy to understand, and beginner-friendly.


Features

Add new tasks
Delete existing tasks
Prevents empty task entries
Clean and responsive UI using Tailwind
Simple and extendable React structure


Tech Stack

React (Functional components + Hooks)
Vite (Fast dev environment)
Tailwind CSS (Utility-first styling)
